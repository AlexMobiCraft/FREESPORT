# Исправление ошибки 405 (Полный цикл)

Я проверил конфигурацию: на вашем сервере код "запекается" внутри контейнера. Простой перезагрузки (`restart`) **НЕДОСТАТОЧНО**, так как она запускает старую версию кода.

Нужно пересобрать контейнер.

Выполните на сервере:

## 1. Обновление кода
```bash
cd /home/freesport/freesport/
git pull
```

## 2. Пересборка и запуск (ОБЯЗАТЕЛЬНО)
Эта команда пересоберет образ с новым кодом (поддержкой POST).

```bash
docker compose --env-file .env.prod -f docker/docker-compose.prod.yml up -d --build backend
```

## 3. Проверка
После того как команда выполнится (может занять 1-2 минуты), попробуйте проверить соединение в 1С.
