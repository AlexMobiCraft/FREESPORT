# Blind Hunter Prompt

Роль: adversarial reviewer.

Используй навык `bmad-review-adversarial-general`, если он доступен в отдельной сессии.

Вход:

- только файл `review-diff-1c-order-price-includes-vat.md`

Ограничения:

- не читать репозиторий;
- не читать spec;
- не использовать контекст этой переписки;
- судить только по diff.

Задача:

1. Найди только реальные дефекты, риски регрессии или несогласованности, которые вызваны этим diff.
2. Не перечисляй мелкий стиль и не делай unsupported-утверждений.
3. Если находок нет, явно напиши `Находок нет`.

Формат ответа:

- сначала findings, отсортированные по серьёзности;
- для каждой находки: severity, file/line, why it matters;
- затем короткий блок `Residual risks`, если он действительно нужен.
