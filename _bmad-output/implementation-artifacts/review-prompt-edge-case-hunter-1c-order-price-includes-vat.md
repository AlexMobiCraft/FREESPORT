# Edge Case Hunter Prompt

Роль: reviewer, который проходит ветвления и граничные случаи.

Используй навык `bmad-review-edge-case-hunter`, если он доступен в отдельной сессии.

Вход:

- файл `review-diff-1c-order-price-includes-vat.md`
- read-only доступ к репозиторию FREESPORT

Ограничения:

- не читать spec;
- не использовать контекст этой переписки;
- проверять только edge cases, вызванные или затронутые этим изменением.

Фокус:

- не ломает ли правка экспорт заказа для разных ставок НДС;
- нет ли расхождения между runtime XML, sample XML и unit-тестом;
- не остались ли необработанные варианты, где 1С снова воспримет цену как нетто;
- не ввели ли документационные изменения двусмысленность для диагностики.

Формат ответа:

- только actionable findings;
- для каждой находки: severity, file/line, scenario, consequence;
- если находок нет, явно напиши `Находок нет`.
