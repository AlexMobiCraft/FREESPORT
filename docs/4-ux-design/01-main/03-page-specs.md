- **Skew angle:** `-12deg` (интерактивные элементы)
- **Counter-skew:** `12deg` (текст внутри skewed контейнеров)

## 5. Стратегия рефакторинга

### Принцип: `variant` prop

Вместо создания новых Electric-компонентов, **рефакторим существующие** с добавлением prop:

```tsx
// До
<HitsSection />

// После
<HitsSection variant="electric" />
```

### План работ

| Компонент | Приоритет | Сложность |
|-----------|-----------|-----------|
| HitsSection | ✅ Done | — |
| NewArrivalsSection | P1 | Low (copy HitsSection pattern) |
| SaleSection | P1 | Low |
| PromoSection | P2 | Low |
| CategoriesSection | P2 | Medium |
| NewsSection | P3 | Medium |
| SubscribeNewsSection | P3 | Medium |
