Проблема: при загрузке заказа с сайта в 1С поля документа ЗаказКлиента — Организация и Склад — оставались пустыми, хотя сайт уже передает эти значения в XML.

  Выводы:                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                               
  - На стороне сайта данные выгружаются: Организация и Склад есть в XML, включая ЗначенияРеквизитов.                                                                                                                                                           
  - Обработчик БУС в 1С не заполняет эти поля напрямую из верхнеуровневых тегов XML.                                                                                                                                                                           
  - Организация по умолчанию настроена как фиксированное значение и может оставаться пустой.                                                                                                                                                                   
  - Склад разбирается из XML, но стандартный алгоритм заполнения поля документа фактически не использует это значение напрямую.                                                                                                                                
  - Рабочее решение — настроить заполнение через свойства документа:                                                                                                                                                                                           
                                                                                                                                                                                                                                                               
  Организация:                                                                                                                                                                                                                                                 
  Тип данных: Сопоставление по свойству документа                                                                                                                                                                                                              
  Значение: Организация                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                               
  Склад:                                                                                                                                                                                                                                                       
  Тип данных: Сопоставление по свойству документа                                                                                                                                                                                                              
  Значение: Склад                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                               
  Важно: значения Организация и Склад, приходящие с сайта, должны точно совпадать с наименованиями объектов в 1С, потому что поиск выполняется по наименованию.
