## 🚀 Введение

**Electric Orange** — это дизайн-система, воплощающая философию "Digital Brutalism & Kinetic Energy". Она построена на контрасте темного фона, ярко-оранжевого акцента и агрессивной геометрии.

### Основные Принципы

1. **Геометрия (Skew):** Интерактивные элементы скошены на **-12 градусов**. Контент внутри контейнеров (например, инпуты, карточки) остается прямоугольным.
2. **Типографика:**
   - **Headers:** `Roboto Condensed`, Uppercase, Black Weight.
   - **Body:** `Inter`, Regular/Medium.
3. **Цвет:** Глубокий черный фон (`#0F0F0F`) и электрический оранжевый (`#FF6B00`).

## 🛠️ Как использовать (для разработчиков)

### 1. Импорт стилей

Дизайн-система подключена глобально через `src/app/globals-electric-orange.css`. Убедитесь, что этот файл импортирован в корневом layout.

### 2. Использование компонентов

Все UI-компоненты находятся в `src/components/ui/`. Используйте готовые компоненты `Electric*` (например, `ElectricButton`, `ElectricInput`), а не создавайте их с нуля.

```tsx
import { ElectricButton } from "@/components/ui/Button/ElectricButton";

<ElectricButton variant="primary">В КОРЗИНУ</ElectricButton>;
```

### 3. Создание новых элементов

Если вам нужно создать новый элемент, следуйте правилам из `01-tokens.md`. Используйте Tailwind утилиты, соответствующие токенам.

- **Скос:** `transform: skewX(-12deg)`
- **Контр-скос текста:** `transform: skewX(12deg)`
- **Цвет:** `text-[var(--color-primary)]` или `bg-[var(--color-primary)]`

## ⚠️ Checklist перед PR

- [ ] Все интерактивные кнопки имеют скос -12deg.
- [ ] Все инпуты ввода текста — прямоугольные.
- [ ] Нет скругленных углов (border-radius: 0).
- [ ] Используется только палитра Electric Orange (никаких синих/зеленых оттенков из старой темы).
- [ ] Анимации плавные и соответствуют токенам.
