# Миграция цветовой схемы FREESPORT на сине-голубую палитру

> **Версия:** 1.0.0  
> **Дата:** 2025-11-25  
> **Статус:** Планирование

---

## 1. Обзор

Принято решение о переходе с текущей **графитово-серой** цветовой схемы на **сине-голубую** палитру для создания более современного, технологичного и дружелюбного визуального стиля в духе OZON/Wildberries.

### Цели миграции
- Создать более яркий и запоминающийся бренд
- Улучшить визуальную иерархию с помощью акцентных цветов
- Повысить конверсию CTA-элементов за счёт контрастных кнопок
- Сохранить читаемость и доступность (WCAG AA)

## 7. Accessibility (Доступность)

### Контрастность текста (WCAG AA требует ≥ 4.5:1)

| Комбинация | Контраст | Статус |
|------------|----------|--------|
| `#0060FF` на `#FFFFFF` | 4.5:1 | ✅ Pass |
| `#1F2A44` на `#FFFFFF` | 12.6:1 | ✅ Pass |
| `#4B5C7A` на `#FFFFFF` | 6.2:1 | ✅ Pass |
| `#FFFFFF` на `#0060FF` | 4.5:1 | ✅ Pass |
| `#0060FF` на `#E7F3FF` | 4.1:1 | ⚠️ Borderline |

### Рекомендации
- Для мелкого текста на `#E7F3FF` использовать `#0047CC` вместо `#0060FF`
- Кнопки с белым текстом на `#0060FF` соответствуют стандарту

## 8. Примечания

1. **Обратная совместимость**: Старые классы Tailwind (`bg-gray-*`) продолжат работать, но рекомендуется переход на токены
2. **CSS-переменные**: Большинство изменений применится автоматически через обновление `globals.css`
3. **Хардкод**: Компоненты с хардкодом цветов требуют ручного обновления
4. **Тени**: Тени уже используют синие оттенки, изменения минимальны
