# Epic 19: Информационные страницы

## Описание

Создание информационных страниц для B2B платформы FREESPORT: «О компании» (/about), «Партнёрам» (/partners), «Доставка» (/delivery), а также интеграция тизер-блоков на главную страницу.

## Бизнес-ценность

- Повышение конверсии B2B посетителей через чёткую коммуникацию условий сотрудничества
- Улучшение SEO за счёт отдельных landing pages для ключевых запросов
- Формирование доверия к бренду через страницу «О компании»
- Снижение нагрузки на менеджеров за счёт FAQ и прозрачных условий

### Story 19.1: Компоненты для информационных страниц
