# Обзор возможностей панели управления FREESPORT (Django Admin)

Данный документ представляет собой обзор административного интерфейса портала FREESPORT, разработанного для эффективного управления B2B и B2C процессами, каталогом товаров и автоматизированными интеграциями.

---

## Общая информация

Админ-панель является "сердцем" портала, предоставляя полный контроль над данными с акцентом на безопасность.

- **Брендирование**: Интерфейс адаптирован под фирменный стиль FREESPORT.
- **Аудит (Audit Log)**: Каждое критичное действие логируется с фиксацией IP-адреса.
- **Безопасность**: Настроен доступ на основе разрешений (Permissions).

## 1. Каталог товаров

### 1.1 Бренды

Управление брендами спортивных товаров.

| Поле | Описание |
| --- | --- |
| Название | Наименование бренда |
| Slug | URL-адрес бренда |
| Маппинги 1С | Количество связей с кодами 1С |
| Активность | Включение/выключение бренда |

**Действия**: Объединение дублирующихся брендов без потери связей с товарами.

---

### 1.2 Товары

Основной каталог продукции с маркетинговыми инструментами.

| Поле | Описание |
| --- | --- |
| Название | Наименование товара |
| Бренд | Связь с брендом |
| Категория | Связь с категорией |
| Маркетинговые флаги | Хит, Новинка, Sale, Акция, Премиум |
| Скидка | Процент скидки |

**Фильтры**: Активность, Бренд, Категория, Статус синхронизации, Маркетинговые флаги.

**Действия**: Массовая установка/снятие флагов Хит, Новинка, Sale, Акция, Премиум.

---

### 1.3 Варианты товаров

Управление SKU с учетом цветов и размеров.

| Поле | Описание |
| --- | --- |
| SKU | Артикул варианта |
| Товар | Связь с родительским товаром |
| Цвет | Название цвета |
| Размер | Значение размера |
| Розничная цена | Цена для B2C |
| Опт 1-3, Тренер, Федерация | Цены для B2B |
| Остаток | Количество на складе |
| Резерв | Зарезервированное количество |

## Заключение

Админ-панель FREESPORT — это полноценное рабочее место менеджера, минимизирующее рутину и исключающее ошибки при работе с B2B-сегментом. Каждый раздел оптимизирован для быстрой работы с данными и поддерживает массовые операции.
