# Эпик 15: Checkout - Изменения в версии 2.0

**Дата обновления:** 2025-12-14
**Причина:** Упрощение эпика под текущий этап разработки

---

## ⚠️ Критические изменения

### **Версия 1.0 → Версия 2.0: Упрощённая реализация**

#### ❌ **Что УДАЛЕНО из scope:**

1. **Расчёт стоимости доставки:**
   - ❌ Интеграция с CDEK API
   - ❌ Интеграция с Boxberry API
   - ❌ Динамический расчёт стоимости на основе адреса и веса
   - ❌ Story 15.3 (старая): "Расчёт стоимости доставки"
   - ❌ `deliveryService.calculateDelivery()`
   - ❌ `/delivery/calculate` API endpoint

2. **Онлайн оплата:**
   - ❌ Интеграция с YuKassa Payment Gateway
   - ❌ Обработка платежей на frontend
   - ❌ Story 15.4 (старая): "Управление способами оплаты"
   - ❌ `paymentService.ts`
   - ❌ `PaymentMethods.tsx` компонент
   - ❌ `/payment/methods` API endpoint

3. **Автоматические статусы заказов:**
   - ❌ Автоматическое подтверждение заказа
   - ❌ Автоматическое изменение статусов
   - ❌ `/orders/confirm` API endpoint

#### ✅ **Что ДОБАВЛЕНО вместо:**

1. **Ручная обработка заказов:**
   - ✅ Заказ создаётся со статусом "Новый"
   - ✅ Email администратору с деталями заказа
   - ✅ Администратор вручную меняет статусы через Django Admin:
     - Новый → Оплачен → Отправлен → Доставлен
   - ✅ Backend отвечает за email отправку

2. **Предустановленные способы доставки:**
   - ✅ Список способов настраивается через Django Admin
   - ✅ Radio group для выбора: Транспортная компания, Курьер, Самовывоз
   - ✅ Стоимость отображается как "Уточняется администратором"
   - ✅ `GET /delivery/methods` API endpoint

3. **Упрощённая форма checkout:**
   - ✅ БЕЗ секции "Способ оплаты"
   - ✅ Секции: Контакты, Адрес, Доставка, Комментарий
   - ✅ Информационный блок на success странице: "Администратор свяжется с вами"
