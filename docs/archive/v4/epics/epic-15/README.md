## Быстрый обзор

**Цель:** Реализовать полноценный процесс оформления заказа (checkout) для платформы FREESPORT с поддержкой B2B/B2C сценариев.

**Ключевые функции:**

- Single Page Checkout интерфейс
- Интеграция с Orders API
- Расчёт доставки (CDEK, Boxberry)
- Управление способами оплаты (YuKassa)
- Страница подтверждения заказа
- E2E тестирование критического флоу

**Зависимости:**

- Эпик 10 (Фундамент) - UI Kit, apiClient, Zustand
- Эпик 13 (Аутентификация) - authStore, JWT
- Эпик 14 (Корзина) - cartStore, товары
