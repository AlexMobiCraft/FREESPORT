# Визуализация монохромной терракотовой палитры теней

## Цветовая палитра

### Основные цвета текста
- **Primary (#1B1B1B)** - глубокий терракотово-черный
- **Secondary (#4D4D4D)** - средний терракотово-серый  
- **Muted (#7A7A7A)** - светлый терракотово-серый
- **Inverse (#FFFFFF)** - белый

## Визуальное представление теней

### 1. Default Shadow
```
box-shadow: 0 8px 24px rgba(27, 27, 27, 0.08)
```
**Применение:** Базовые карточки, панели контента
**Визуально:** Легкая, едва заметная тень создающая глубину

### 2. Hover Shadow  
```
box-shadow: 0 10px 32px rgba(27, 27, 27, 0.12)
```
**Применение:** Интерактивные элементы при наведении
**Визуально:** Более выраженная тень, элемент "поднимается"

### 3. Primary Shadow
```
box-shadow: 0 10px 24px rgba(77, 77, 77, 0.28)
```
**Применение:** Кнопки CTA, важные интерактивные элементы
**Визуально:** Яркая, заметная тень для привлечения внимания

### 4. Secondary Shadow
```
box-shadow: 0 6px 16px rgba(77, 77, 77, 0.12)
```
**Применение:** Вторичные кнопки, менее важные элементы
**Визуально:** Умеренная тень для создания иерархии

### 5. Pressed Shadow
```
box-shadow: 0 6px 18px rgba(77, 77, 77, 0.24)
```
**Применение:** Нажатые состояния кнопок, активные элементы
**Визуально:** Глубокая тень, создающая ощущение "нажатия"

### 6. Modal Shadow
```
box-shadow: 0 24px 64px rgba(27, 27, 27, 0.24)
```
**Применение:** Модальные окна, выпадающие меню
**Визуально:** Очень глубокая тень для отделения от фона

## Сравнение с текущими синими тенями

### Текущие (противоречат концепции):
- `rgba(0, 96, 255, 0.28)` - яркий синий
- `rgba(0, 71, 204, 0.12)` - средний синий  
- `rgba(0, 55, 166, 0.24)` - темно-синий

### Новые (гармоничны с концепцией):
- `rgba(27, 27, 27, 0.08)` - темный серый
- `rgba(77, 77, 77, 0.12)` - средний серый
- `rgba(77, 77, 77, 0.24)` - насыщенный серый

## Визуальные примеры использования

### Карточка товара
```
background: #FFFFFF
border-radius: 12px
box-shadow: 0 8px 24px rgba(27, 27, 27, 0.08)
```
**Результат:** Чистая белая карточка с легкой серой тенью

### Кнопка "Добавить в корзину"
```
background: #1B1B1B
color: #FFFFFF
border-radius: 16px
box-shadow: 0 10px 24px rgba(77, 77, 77, 0.28)
```
**Результат:** Темная кнопка с выразительной серой тенью

### Модальное окно
```
background: #FFFFFF
border-radius: 24px
box-shadow: 0 24px 64px rgba(27, 27, 27, 0.24)
```
**Результат:** Светлое модальное окно с глубокой серой тенью

## Преимущества новой палитры

1. **Консистентность:** Все тени используют ту же цветовую базу, что и текст
2. **Монохромность:** Соответствие стилю Nike с акцентом на контрасты
3. **Доступность:** Лучше читаемость на различных фонах
4. **Иерархия:** Четкое визуальное разделение элементов по важности
5. **Современность:** Соответствие трендам на минимализм и монохромные интерфейсы

## Рекомендации по внедрению

1. **Постепенный переход:** Заменять тени поэтапно, начиная с основных компонентов
2. **Тестирование:** Проверить читаемость на различных устройствах и в разных условиях освещения
3. **Адаптивность:** Убедиться, что тени хорошо выглядят на мобильных устройствах
4. **Контроль качества:** Проверить соответствие с WCAG стандартами доступности
