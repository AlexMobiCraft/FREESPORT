# Инструкция по обновлению теней в design-system.json

## Обзор

Необходимо заменить синие тени в секции `shadows` (строки 151-156) на монохромную терракотовую палитру в соответствии с новой концепцией дизайна в стиле Nike.

## Текущие значения для замены

```json
"shadows": {
  "default": "0 8px 24px rgba(15, 23, 42, 0.08)",
  "hover": "0 10px 32px rgba(15, 23, 42, 0.12)",
  "primary": "0 10px 24px rgba(0, 96, 255, 0.28)",
  "secondary": "0 6px 16px rgba(0, 71, 204, 0.12)",
  "pressed": "0 6px 18px rgba(0, 55, 166, 0.24)",
  "modal": "0 24px 64px rgba(15, 23, 42, 0.24)"
}
```

## Новые значения

```json
"shadows": {
  "default": "0 8px 24px rgba(27, 27, 27, 0.08)",
  "hover": "0 10px 32px rgba(27, 27, 27, 0.12)",
  "primary": "0 10px 24px rgba(77, 77, 77, 0.28)",
  "secondary": "0 6px 16px rgba(77, 77, 77, 0.12)",
  "pressed": "0 6px 18px rgba(77, 77, 77, 0.24)",
  "modal": "0 24px 64px rgba(27, 27, 27, 0.24)"
}
```

## Дополнительные тени (рекомендуется добавить)

```json
"focus": "0 0 0 2px rgba(77, 77, 77, 0.2)",
"success": "0 4px 12px rgba(122, 122, 122, 0.15)"
```

## Пошаговая инструкция по обновлению

### Шаг 1: Откройте файл design-system.json

Расположение: `frontend/docs/design-system.json`

### Шаг 2: Найдите секцию shadows (строки 151-156)

Ищите блок:
```json
"shadows": {
  "default": "...",
  "hover": "...",
  // и т.д.
}
```

### Шаг 3: Замените значения

Замените каждое значение на новое из таблицы выше:

1. **default** - замените `rgba(15, 23, 42, 0.08)` на `rgba(27, 27, 27, 0.08)`
2. **hover** - замените `rgba(15, 23, 42, 0.12)` на `rgba(27, 27, 27, 0.12)`
3. **primary** - замените `rgba(0, 96, 255, 0.28)` на `rgba(77, 77, 77, 0.28)`
4. **secondary** - замените `rgba(0, 71, 204, 0.12)` на `rgba(77, 77, 77, 0.12)`
5. **pressed** - замените `rgba(0, 55, 166, 0.24)` на `rgba(77, 77, 77, 0.24)`
6. **modal** - замените `rgba(15, 23, 42, 0.24)` на `rgba(27, 27, 27, 0.24)`

### Шаг 4: Добавьте новые тени (опционально)

После существующих теней добавьте:

```json
"focus": "0 0 0 2px rgba(77, 77, 77, 0.2)",
"success": "0 4px 12px rgba(122, 122, 122, 0.15)"
```

### Шаг 5: Обновите brand.tone (строка 11)

Замените:
```json
"tone": "Технологичный, дружелюбный e-commerce в стиле Nike"
```

### Шаг 6: Проверьте другие секции

Убедитесь, что в других секциях нет синих оттенков, которые противоречат новой концепции:

- Проверьте `iconography.defaultColor` (строка 238)
- Проверьте `interaction.focus.outlineColor` (строка 255)
- Проверьте `interaction.hover.colorShift` (строка 252)

### Шаг 7: Валидация JSON

После изменений проверьте валидность JSON-файла:
```bash
# Можно использовать онлайн-валидатор или команду
cat frontend/docs/design-system.json | python -m json.tool
```

## Результат

После применения этих изменений дизайн-система будет соответствовать монохромной терракотовой концепции в стиле Nike с гармоничными тенями, создающими глубину и иерархию без использования цветовых акцентов.
