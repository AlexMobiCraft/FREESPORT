## Acceptance Criteria

### AC 1: Автоматизированное тестирование

- [ ] `npm test` - все unit тесты проходят (100%)
- [ ] `npm run test:update-snapshots` - snapshots обновлены
- [ ] `npm run test:coverage` - coverage >= 80%
- [ ] Нет console errors в тестах

### AC 2: Accessibility Audit

- [ ] `npm run test:a11y` - axe-core audit пройден
- [ ] Контрастность WCAG AA:
  - [ ] #0060FF на #FFFFFF = 4.5:1 ✅
  - [ ] #1F2A44 на #FFFFFF = 12.6:1 ✅
  - [ ] #4B5C7A на #FFFFFF = 6.2:1 ✅
- [ ] Focus indicators видимы

### AC 3: Visual Regression Testing

Проверить на 3 breakpoints (Desktop 1920px, Tablet 768px, Mobile 375px):

- [ ] Главная страница
- [ ] Каталог
- [ ] Карточка товара
- [ ] Корзина
- [ ] Checkout
- [ ] Личный кабинет

### AC 4: Manual QA Testing

Браузеры: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

- [ ] Все интерактивные состояния работают (hover, focus, active, disabled)
- [ ] Формы корректны (validation, submission)
- [ ] Навигация без ошибок
- [ ] Функциональность без регрессии

### AC 5: Performance Testing

- [ ] PageSpeed >70 на главных страницах
- [ ] Bundle size не увеличился
- [ ] LCP < 2.5s
- [ ] No layout shifts (CLS < 0.1)

### AC 6: Deployment

- [ ] Staging deployment успешен
- [ ] Staging smoke tests пройдены
- [ ] Stakeholder approval получен
- [ ] Production deployment выполнен
- [ ] Production monitoring (24 часа): нет критических ошибок
