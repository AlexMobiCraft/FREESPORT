## Acceptance Criteria

### AC 1: Catalog Page — Pagination Number Buttons

**Файл:** `frontend/src/app/catalog/page.tsx`
**Строки:** 726-738

**БЫЛО (строка 732):**

```tsx
pageNumber === page
  ? "h-10 w-10 rounded-full bg-blue-600 text-white"
  : "h-10 w-10 rounded-full border border-gray-200 text-gray-600";
```

**СТАЛО:**

```tsx
pageNumber === page
  ? "h-10 w-10 rounded-full bg-primary text-white hover:bg-primary-hover"
  : "h-10 w-10 rounded-full border border-neutral-300 text-neutral-600 hover:border-primary hover:text-primary";
```

**Критерии приёмки:**

- [x] `bg-blue-600` заменён на `bg-primary` (#0060FF)
- [x] Добавлен `hover:bg-primary-hover` для активной кнопки
- [x] `border-gray-200` заменён на `border-neutral-300`
- [x] `text-gray-600` заменён на `text-neutral-600`
- [x] Добавлены hover-состояния для неактивных кнопок

### Токены дизайн-системы v2.1.0

**Источники токенов:**

- **design-system.json:** `frontend/design-system.json` → `colors.primary`, `colors.accent.danger`, `colors.neutral.*`
- **globals.css:** `frontend/src/app/globals.css` → CSS-переменные `--color-primary`, `--color-accent-danger`, etc.

**После выполнения Story 14.1, токены в `globals.css` будут:**

| Токен              | HEX       | Ключ в design-system.json      | Применение                     |
| ------------------ | --------- | ------------------------------ | ------------------------------ |
| `primary`          | `#0060FF` | `colors.primary.DEFAULT`       | Активная pagination, CTA       |
| `primary-hover`    | `#0047CC` | `colors.primary.hover`         | Hover состояние                |
| `accent-danger`    | `#E53935` | `colors.accent.danger.DEFAULT` | Текст badge (sale, cart)       |
| `accent-danger-bg` | `#FFE1E8` | `colors.accent.danger.bg`      | Фон badge (sale, cart)         |
| `neutral-300`      | `#E3E8F2` | `colors.neutral.300`           | Borders неактивных элементов   |
| `neutral-500`      | `#8A9BB8` | `colors.neutral.500`           | Текст navigation buttons (← →) |
| `neutral-600`      | `#6B7A99` | `colors.neutral.600`           | Текст неактивных элементов     |

### Coding Standards

**Формат Tailwind классов:**

```tsx
// ✅ Правильно - используем токены
className = "bg-primary text-white hover:bg-primary-hover";

// ❌ Неправильно - хардкод Tailwind цветов
className = "bg-blue-600 text-white hover:bg-blue-700";

// ⚠️ Допустимо - хардкод HEX для фона badge (нет токена)
className = "bg-[#FFE1E8] text-accent-danger";
```

### Важные замечания

1. **Зависимость от Story 14.1:** Токены `primary`, `accent-danger`, `accent-danger-bg` и т.д. должны быть обновлены в `globals.css` ПЕРЕД выполнением этой истории
2. **Три места для cart badge:** Не забыть обновить ВСЕ три места (B2B, desktop cart, mobile cart)
3. **Hover состояния:** Добавить hover для улучшения UX
4. **Navigation buttons:** Кнопки ← и → также должны использовать токены
