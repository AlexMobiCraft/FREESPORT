## Story

**As a** администратор системы,
**I want** чтобы при выборе типа импорта "Изображения товаров" запускалась Celery задача, которая обновляет изображения всех товаров из директории 1С,
**so that** я могу массово обновить изображения товаров асинхронно с отслеживанием прогресса через ImportSession.

---

## Acceptance Criteria

1. Задача запускается асинхронно при выборе типа "Изображения"
2. Прогресс обновляется в `ImportSession.report_details`
3. Статистика содержит: `total_products`, `processed`, `copied`, `skipped`, `errors`
4. При ошибках сессия помечается как `failed` с детальным `error_message`
5. Задача использует существующий метод `ProductDataProcessor.import_product_images()`
6. Поддерживается chunked processing (по 100 товаров за итерацию)
7. Директория `import_files/` проверяется перед началом обработки
8. Задача интегрируется с существующей системой `_execute_import_type()`

**Логика поиска изображений для товара**:

```python
from pathlib import Path

def get_product_images(product: Product, base_dir: str) -> list[str]:
    """
    Получить список изображений для товара из директории 1С.

    Args:
        product: Product instance с onec_id
        base_dir: Путь к goods/import_files/

    Returns:
        Список относительных путей (например, ["00/001a16a4_image.jpg"])
    """
    # Первые 2 символа onec_id определяют поддиректорию
    onec_id = product.onec_id
    subdir = onec_id[:2] if len(onec_id) >= 2 else "00"

    images_dir = Path(base_dir) / subdir

    if not images_dir.exists():
        return []

    # Поиск файлов начинающихся с onec_id
    image_paths = []
    for ext in ["*.jpg", "*.jpeg", "*.png"]:
        for img_file in images_dir.glob(f"{onec_id}*{ext}"):
            # Относительный путь от base_dir
            relative_path = f"{subdir}/{img_file.name}"
            image_paths.append(relative_path)

    return image_paths
```

**4. Паттерн обновления ImportSession**:

```python
from django.utils import timezone
from apps.products.models import ImportSession

def _run_image_import(task_id: str) -> dict[str, str]:
    """Импорт изображений товаров из директории 1С."""

    # Найти сессию по task_id
    try:
        session = ImportSession.objects.get(celery_task_id=task_id)
    except ImportSession.DoesNotExist:
        logger.warning(f"ImportSession не найдена для task_id: {task_id}")
        session = None

    # Обновить статус на IN_PROGRESS
    if session:
        session.status = ImportSession.ImportStatus.IN_PROGRESS
        session.save(update_fields=["status"])

    try:
        # Основная логика импорта
        total_products = Product.objects.filter(is_active=True).count()
        processed = 0
        total_copied = 0
        total_skipped = 0
        total_errors = 0

        # ... обработка товаров ...

        # Обновление прогресса
        if session and processed % 50 == 0:
            session.report_details = {
                "total_products": total_products,
                "processed": processed,
                "copied": total_copied,
                "skipped": total_skipped,
                "errors": total_errors,
                "last_updated": timezone.now().isoformat()
            }
            session.save(update_fields=["report_details"])

        # Финализация
        if session:
            session.status = ImportSession.ImportStatus.COMPLETED
            session.finished_at = timezone.now()
            session.report_details = {
                "total_products": total_products,
                "processed": processed,
                "copied": total_copied,
                "skipped": total_skipped,
                "errors": total_errors,
                "completed_at": timezone.now().isoformat()
            }
            session.save(update_fields=["status", "finished_at", "report_details"])

        return {"type": "images", "message": f"Обработано {processed} товаров"}

    except Exception as e:
        logger.error(f"Ошибка импорта изображений: {e}", exc_info=True)

        if session:
            session.status = ImportSession.ImportStatus.FAILED
            session.finished_at = timezone.now()
            session.error_message = str(e)
            session.save(update_fields=["status", "finished_at", "error_message"])

        raise
```

### Важные замечания

1. **Story 15.2 НЕ включает изменения в views.py**:
   - Story 15.1 уже добавила маппинг в `session_type_map`
   - Celery задача автоматически вызовется при выборе типа "images"

   - Для 1000 товаров с по 3 изображения: ~5-10 минут

   - Chunked processing предотвращает OOM
   - Обновление сессии каждые 50 товаров снижает нагрузку на БД

3. **Обработка edge cases**:
   - Товар без `onec_id` - пропустить (skipped++)
   - Директория товара не существует - пропустить (skipped++)
   - Нет изображений в директории - пропустить (skipped++)

   - Новая ветка в `_execute_import_type()` не влияет на catalog, stocks, prices, customers
   - Формат результата `{"type": "images", "message": "..."}` совместим с существующим

   - По умолчанию `validate_images=False` для производительности
