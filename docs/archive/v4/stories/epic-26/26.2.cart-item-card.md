## Story

**Как** пользователь,  
**Я хочу** видеть информацию о каждом товаре в корзине и управлять его количеством,  
**Чтобы** редактировать свой заказ перед оформлением.

## Acceptance Criteria

1. Отображается изображение товара (88x88px, border-radius: 12px)
2. Отображается название товара, артикул (SKU), цвет, размер
3. Отображается цена за единицу и сумма по позиции
4. Кнопки +/- для изменения количества товара
5. Input для ручного ввода количества
6. Кнопка удаления товара (иконка 🗑)
7. Optimistic Updates при изменении quantity
8. Валидация quantity: min=1, max=99 (stock недоступен в CartItem)
9. Toast уведомление при удалении товара
10. Disabled state для +/- кнопок при min/max quantity
11. Loading indicator при обновлении quantity (spinner на кнопке)
12. Image placeholder при отсутствии изображения
13. Accessibility: aria-labels, keyboard navigation, role="spinbutton"

<div

        {item.variant.size_value && ` | Размер: ${item.variant.size_value}`}

interface CartItem {
  id: number;
  variant_id: number;
  product: {
    id: number;
    name: string;
    slug: string;
    image: string | null; // может быть null!
  };
  variant: {
    sku: string;
    color_name: string | null;
    size_value: string | null;
  };
  quantity: number;
  unit_price: string; // ⚠️ STRING! Используй parseFloat()
  total_price: string; // ⚠️ STRING! Используй parseFloat()
  added_at: string;
}

### Optimistic Updates Pattern

```typescript
const handleQuantityChange = async (itemId: number, newQuantity: number) => {
  // Optimistic update уже реализован в cartStore
  // Используем updateQuantity (строка 20 в cartStore.ts)
  const result = await updateQuantity(itemId, newQuantity);

  if (!result.success) {
    toast.error("Не удалось обновить количество");
    // Rollback выполняется автоматически в cartStore
  }
};

const handleRemove = async (itemId: number) => {
  const result = await removeItem(itemId);

  if (result.success) {
    toast.success("Товар удалён из корзины");
  } else {
    toast.error("Не удалось удалить товар");
  }
};
```

## Testing

**Framework:** Vitest + React Testing Library

**Тестовые сценарии:**

1. **Render:** Отображение всех полей товара
2. **Quantity increment:** Клик на "+" → quantity увеличивается
3. **Quantity decrement:** Клик на "-" → quantity уменьшается
4. **Quantity bounds:** "-" disabled при quantity=1, "+" disabled при max
5. **Remove item:** Клик на 🗑 → вызов onRemove
6. **Price calculation:** total_price = unit_price × quantity
7. **Image fallback:** Placeholder при отсутствии изображения
8. **Debounce input:** Ручной ввод quantity → API вызов происходит через 300ms (использовать `vi.useFakeTimers()` и `vi.advanceTimersByTime(300)`)
